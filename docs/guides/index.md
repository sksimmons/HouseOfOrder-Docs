# Guides

These guides describe **HouseOfOrder the product**, not a specific operating system. The same workflows apply on every supported client.

Add topics as you need them, for example:

- Getting started for a new coordinator
- Importing a roster from a spreadsheet
- Generating and editing assignments
- Marking attendance and using history
- Backing up and restoring app data
- Email drafts and notifications

If a step truly differs on one platform, call it out inline; otherwise keep one path for everyone.
