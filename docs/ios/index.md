# HouseOfOrder for iOS

HouseOfOrder helps church cleaning coordinators manage families, generate assignments, track attendance, and draft notification emails.

This section is maintained **in the iOS repository** under `Docs/` and copied into the public documentation site when you run the sync script (see the docs repo README).

## Audience

- Coordinators using the app day to day
- Future contributors to the iOS (and Android) codebases
